## 🐛 常见问题排查

### 问题 1: 翻译失败 - CORS 错误

**症状**：
```
Access to fetch at 'http://localhost:8000/api/v1/translation/translate' 
from origin 'http://localhost:3000' has been blocked by CORS policy
```

**解决方案**：
1. 检查后端 CORS 配置（已修复）
2. 重启后端服务器
3. 清除浏览器缓存

### 问题 2: 翻译失败 - 401/403 错误

**症状**：
```
401 Unauthorized 或 403 Forbidden
```

**解决方案**：
1. 检查是否已登录
2. 检查 localStorage 中的 authToken
3. 重新登录

### 问题 3: 文档上传失败

**症状**：
- 上传后无响应
- 显示 500 错误

**解决方案**：
1. 检查后端日志
2. 验证文件格式
3. 检查 Google Gemini API 密钥配置

### 问题 4: AI 元数据生成失败

**症状**：
- 标题、摘要、分类为空
- 显示 "AI generation failed"

**解决方案**：
1. 检查 `.env` 文件中的 `GOOGLE_API_KEY`
2. 验证 API 密钥是否有效
3. 检查 API 配额
