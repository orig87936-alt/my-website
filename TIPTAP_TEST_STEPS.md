### **步骤 5: 测试富文本编辑器**

#### **5.1 验证编辑器显示**
- [ ] 看到"中文内容"编辑器
- [ ] 看到"英文内容"编辑器
- [ ] 编辑器有工具栏（撤销、重做、标题、粗体等按钮）
- [ ] 编辑器有占位符提示文字

#### **5.2 测试工具栏按钮**

**标题测试**:
1. 在编辑器中输入一些文字
2. 选中文字
3. 点击 H1 按钮
4. 验证文字变成一级标题（大字体）
5. 点击 H2 按钮
6. 验证文字变成二级标题（中等字体）

**粗体测试**:
1. 输入一些文字
2. 选中文字
3. 点击粗体按钮（B）
4. 验证文字变粗

**斜体测试**:
1. 输入一些文字
2. 选中文字
3. 点击斜体按钮（I）
4. 验证文字变斜

**列表测试**:
1. 输入一些文字
2. 点击无序列表按钮（•）
3. 验证文字变成列表项
4. 按 Enter 创建新列表项
5. 验证可以创建多个列表项

**引用测试**:
1. 输入一些文字
2. 点击引用按钮（"）
3. 验证文字变成引用样式（左侧有蓝色竖线）

**代码块测试**:
1. 点击代码块按钮（{}）
2. 输入一些代码
3. 验证代码显示在黑色背景中

#### **5.3 测试 Markdown 语法**

在编辑器中输入以下内容，验证自动转换：

```markdown
# 这是一级标题

## 这是二级标题

这是普通段落。

**这是粗体文字**

*这是斜体文字*

- 列表项 1
- 列表项 2
- 列表项 3

> 这是一段引用

`这是行内代码`
```

验证：
- [ ] `#` 自动转换为 H1
- [ ] `##` 自动转换为 H2
- [ ] `**粗体**` 自动转换为粗体
- [ ] `*斜体*` 自动转换为斜体
- [ ] `- 列表` 自动转换为列表
- [ ] `> 引用` 自动转换为引用
- [ ] `` `代码` `` 自动转换为行内代码

#### **5.4 测试图片上传**

**方式 1: 点击图片按钮**
1. 点击工具栏的图片按钮（🖼️）

3. 等待上传完成

**方式 2: 使用 Markdown 语法**
1. 输入: `![测试图片](图片URL)`
2. 验证图片显示

#### **5.5 测试链接**
1. 输入一些文字
2. 选中文字
3. 点击链接按钮（🔗）
4. 输入链接地址: `https://www.example.com`

#### **5.6 测试撤销/重做**
1. 输入一些文字
2. 点击撤销按钮（↩️）
3. 验证文字消失
4. 点击重做按钮（↪️）

## 🐛 常见问题

### **问题 1: 编辑器不显示**
**可能原因**:
- 依赖未正确安装
- 浏览器缓存问题

**解决方案**:
1. 刷新页面（Ctrl+F5）
2. 清除浏览器缓存
3. 检查浏览器控制台错误

### **问题 2: 图片上传失败**
**可能原因**:
- 后端服务器未运行
- 图片文件过大
- 网络问题

**解决方案**:
1. 检查后端服务器: `http://localhost:8000`
2. 使用较小的图片文件（< 5MB）
3. 检查网络请求

### **问题 3: 工具栏按钮不工作**
**可能原因**:
- JavaScript 错误
- 浏览器兼容性问题

**解决方案**:
1. 检查浏览器控制台错误
2. 使用最新版本的 Chrome/Firefox
3. 刷新页面
