# 📝 文章保存问题解决方案

## 🔍 问题诊断（已更新）

你的文章无法保存到数据库的**真正原因**是：**后端服务器未运行** ❌

### 错误信息分析

从浏览器控制台看到的错误：
- ❌ `Access to fetch at 'http://localhost:8000/api/v1/articles' has been blocked by CORS policy`
- ❌ `POST http://localhost:8000/api/v1/articles net::ERR_FAILED 500`
- ❌ `Failed to fetch article manus-ai from API`
- ❌ `Failed to create article`
- ❌ `Failed to save article`

### 根本原因（按优先级）

1. **后端服务器未运行** ⚠️ **最关键**
   - 后端服务器进程已停止或崩溃
   - 无法连接到 `http://localhost:8000`
   - 导致所有 API 请求失败

2. **创建文章需要管理员权限**
   - 后端 API `/api/v1/articles` (POST) 需要管理员认证
   - 请求头必须包含有效的 Bearer Token
   - 代码位置：`backend/app/routers/articles.py` 第 123-135 行

3. **CORS 错误是表象**
   - 当后端服务器未运行时，浏览器会显示 CORS 错误
   - 实际上是因为无法连接到后端

## 🔧 技术细节

### 认证流程

1. **登录时**：
   ```typescript
   // src/services/api.ts
   export function setAuthToken(token: string): void {
     localStorage.setItem('authToken', token);
   }
   ```

2. **API 请求时**：
   ```typescript
   function buildHeaders(includeAuth: boolean = false): HeadersInit {
     const headers: HeadersInit = {
       'Content-Type': 'application/json',
     };
     
     if (includeAuth) {
       const token = getAuthToken();
       if (token) {
         headers['Authorization'] = `Bearer ${token}`;
       }
     }
     
     return headers;
   }
   ```

3. **后端验证**：
   ```python
   # backend/app/routers/articles.py
   @router.post("", response_model=ArticleResponse, status_code=201)
   async def create_article(
       article_data: ArticleCreate,
       db: AsyncSession = Depends(get_db),
       current_user: User = Depends(require_admin)  # ← 需要管理员权限
   ) -> ArticleResponse:
       article = await article_service.create_article(db, article_data)
       return ArticleResponse.model_validate(article)
   ```

### 数据验证要求

后端对文章数据有以下验证要求：

```python
# backend/app/schemas/article.py
class ArticleCreate(ArticleBase):
    content_zh: List[ContentBlock] = Field(..., description="Chinese content blocks")
    content_en: List[ContentBlock] = Field(..., description="English content blocks")
    
    @field_validator('category')
    @classmethod
    def validate_category(cls, v):
        allowed_categories = ['headline', 'regulatory', 'analysis', 'business', 'enterprise', 'outlook']
        if v not in allowed_categories:
            raise ValueError(f"category must be one of {allowed_categories}")
        return v
```

## 🐛 常见问题

### Q1: 登录后仍然无法保存？

**A**: 检查浏览器控制台：
```javascript
// 在浏览器控制台运行
console.log('Auth Token:', localStorage.getItem('authToken'));
```

如果返回 `null`，说明登录未成功，请重新登录。

### Q2: 提示 "Validation error"？

**A**: 检查文章数据是否符合要求：
- 标题和摘要不能为空
- 摘要至少 20 个字符
- 分类必须是允许的值之一

### Q3: 提示 "Admin privileges required"？

**A**: 当前登录的账号不是管理员，请使用 `admin` 账号登录。

### Q4: 如何查看保存的文章？

**A**: 
1. 访问新闻管理页面：http://localhost:3000/admin/news
2. 或访问 API 文档：http://localhost:8000/api/docs
3. 使用 GET `/api/v1/articles` 端点查看所有文章
