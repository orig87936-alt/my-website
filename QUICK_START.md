## 📝 下一步：5 个简单命令

### 1️⃣ `/speckit.specify` - 描述你想要什么

**在 GitHub Copilot Chat 中输入：**

```
/speckit.specify 

创建一个现代化的个人主页，包含：
- 首屏：大标题、个人介绍、社交媒体链接
- 关于我：详细介绍、技能列表、照片
- 项目展示：卡片式布局，展示我的作品
- 联系方式：联系表单或联系信息
- 导航栏：固定顶部，平滑滚动

设计风格：现代、简洁、支持明暗主题切换
```

---

### 2️⃣ `/speckit.clarify` - 澄清细节（可选）

```
/speckit.clarify
```

AI 会问你问题来确保需求清晰。

---

### 3️⃣ `/speckit.plan` - 指定技术栈

```
/speckit.plan

技术栈：
- React 18 + TypeScript + Vite
- Tailwind CSS 用于样式
- Framer Motion 用于动画
- 响应式设计，移动优先
- 部署到 Vercel
```

---

### 4️⃣ `/speckit.tasks` - 生成任务列表

```
/speckit.tasks
```

自动生成详细的实现任务。

---

### 5️⃣ `/speckit.implement` - 开始实现

```
/speckit.implement
```

AI 会按照计划自动实现功能！

---

## 🎯 可选增强命令

- `/speckit.analyze` - 一致性检查（在 tasks 和 implement 之间）
- `/speckit.checklist` - 质量检查清单（在 plan 之后）
