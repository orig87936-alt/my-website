## 🚀 启动服务

### 1. 启动后端服务

```powershell
# 进入后端目录
cd backend

# 激活虚拟环境
.\venv\Scripts\activate

# 启动服务
uvicorn app.main:app --reload --port 8000
```

**验证后端**：访问 http://localhost:8000/docs 查看 API 文档

### 2. 启动前端服务

```powershell
# 在项目根目录
npm run dev
```

**验证前端**：访问 http://localhost:3000

## 📝 测试步骤

### 步骤 1：登录管理员账户

1. 打开浏览器访问 http://localhost:3000
2. 使用管理员账户登录：
   - 用户名：`admin`
   - 密码：（你的管理员密码）

### 步骤 2：访问新闻管理页面

1. 点击右上角的菜单按钮（三条横线）
2. 在侧边栏中找到 **"管理"** 部分
3. 点击 **"新闻管理 / News Admin"**
4. 你应该看到新闻管理主页面，显示所有文章列表

### 步骤 3：创建新文章

1. 点击右上角的 **"+ 创建新文章"** 按钮
2. 填写表单：
   - **模块分类**：选择一个模块（例如：HEADLINE / 头条新闻）
   - **中文标题**：输入中文标题
   - **英文标题**：输入英文标题
   - **中文摘要**：输入 50-80 字符的中文摘要
   - **英文摘要**：输入 50-80 字符的英文摘要
   - **文章图片**：
     - 点击上传区域或拖拽图片文件
     - 支持 JPG、PNG、WebP 格式
     - 最大 5MB
   - **作者**：输入作者名称（可选）
   - **中文导语**：输入中文导语（可选）
   - **英文导语**：输入英文导语（可选）
   - **中文内容**：输入文章内容，支持简单 Markdown：
     ```
     # 一级标题
     ## 二级标题
     普通段落文本
     ```
   - **英文内容**：输入英文内容

3. 选择发布方式：
   - **保存为草稿**：文章不会在前台显示
   - **发布文章**：文章立即在前台显示

4. 点击按钮提交

### 步骤 4：验证文章创建成功

1. 表单关闭后，你应该在文章列表中看到新创建的文章
2. 文章卡片显示：
   - 缩略图（如果上传了图片）
   - 模块徽章（带颜色）
   - 状态徽章（草稿/已发布）
   - 标题
   - 作者和发布时间
   - 摘要
   - 操作按钮（查看、编辑、删除）

### 步骤 5：筛选和搜索

1. **按模块筛选**：
   - 使用第一个下拉菜单选择模块
   - 列表只显示该模块的文章

2. **按状态筛选**：
   - 使用第二个下拉菜单选择状态
   - 列表只显示该状态的文章

3. **搜索**：
   - 在搜索框输入关键词
   - 列表实时过滤匹配的文章

### 步骤 6：查看文章

1. 点击文章卡片上的 **"查看"** 按钮
2. 跳转到文章详情页
3. 验证文章内容正确显示

### 步骤 7：验证相关文章功能

1. 在文章详情页滚动到底部
2. 查看 **"相关文章"** 区域
3. 应该显示同模块下的其他已发布文章
4. 如果该模块下只有一篇文章，相关文章区域应该为空或显示提示

### 步骤 8：编辑文章

1. 返回新闻管理页面
2. 点击文章卡片上的 **"编辑"** 按钮
3. 跳转到文章详情页（带编辑功能）
4. 修改文章内容
5. 保存更改

### 步骤 9：删除文章

1. 返回新闻管理页面
2. 点击文章卡片上的 **"删除"** 按钮
3. 确认删除对话框
4. 文章从列表中消失

### 步骤 10：测试多模块文章

1. 为每个模块创建至少一篇文章：
   - HEADLINE / 头条新闻
   - REGULATORY BILLS / 监管法规
   - ANALYSIS REPORTS / 分析报告
   - BUSINESS CHANGE / 商业动态
   - CORE ENTERPRISE / 核心企业
   - FUTURE OUTLOOK / 未来展望

2. 验证每个模块的相关文章功能正常工作

## 🐛 常见问题

### 问题 1：无法访问新闻管理页面

**原因**：不是管理员账户

**解决**：
1. 确保使用管理员账户登录
2. 检查用户角色是否为 `admin`

### 问题 2：图片上传失败

**原因**：
- 文件格式不支持
- 文件大小超过 5MB
- 后端服务未运行

**解决**：
1. 检查图片格式（只支持 JPG、PNG、WebP）
2. 压缩图片到 5MB 以下
3. 确认后端服务正在运行

### 问题 3：文章创建后不显示

**原因**：
- 文章保存为草稿状态
- 前端未刷新

**解决**：
1. 检查文章状态（草稿不会在前台显示）
2. 刷新页面

### 问题 4：相关文章为空

**原因**：该模块下只有一篇文章或没有其他已发布文章

**解决**：为该模块创建更多已发布状态的文章
