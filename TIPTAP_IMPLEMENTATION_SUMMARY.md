### **图片上传**

**实现方式**:
```typescript
onImageUpload={async (file) => {
  const formData = new FormData();
  formData.append('file', file);
  const response = await uploadAPI.uploadImage(formData);
  return response.url;
}}
```

**功能**:
- ✅ 点击图片按钮选择文件
- ✅ 自动上传到服务器
- ✅ 返回图片 URL
- ✅ 自动插入到编辑器
- ✅ 支持 URL 输入（备选方案）

## 🔧 技术架构

### **数据流程**

```
用户输入
  ↓
TipTap 编辑器
  ↓
Markdown 文本
  ↓
textToContentBlocks()
  ↓
ContentBlock[] 数组
  ↓
POST /api/v1/articles
  ↓
PostgreSQL JSONB
  ↓
GET /api/v1/articles/{id}
  ↓
MarkdownRenderer
  ↓
显示给用户
```

### **完全兼容现有系统**

- ✅ **后端 API** - 无需修改
- ✅ **数据库 Schema** - 无需修改
- ✅ **ContentBlock 结构** - 完全兼容
- ✅ **MarkdownRenderer** - 正常工作

## 🚀 如何使用

### **1. 访问新闻管理后台**
```
http://localhost:3000
→ 管理员登录
→ 新闻管理
→ 创建新文章
```

### **2. 使用富文本编辑器**
- 方式 1: 点击工具栏按钮
- 方式 2: 输入 Markdown 语法
- 方式 3: 使用快捷键

### **3. 上传图片**
- 点击图片按钮 🖼️
- 选择本地图片
- 自动上传并插入

### **4. 保存文章**
- 点击"发布"或"保存草稿"
- 内容自动转换为 ContentBlock[]
- 发送到后端保存
