## 🎯 核心特点

### 1. 完全独立
- ❌ **不影响** 现有的 `mobile.css`
- ✅ **可随时** 启用/禁用
- ✅ **用户可控** 通过右下角按钮

### 2. 智能缩放
| 设备 | 屏幕宽度 | 缩放比例 |
|------|---------|---------|
| 小屏手机 | ≤480px | 0.65x |
| 大屏手机 | 481-768px | 0.75x |
| 手机横屏 | ≤768px | 0.85x |
| 平板竖屏 | 769-1024px | 0.85x |
| 平板横屏 | 769-1280px | 0.95x |

### 3. 工作原理
```css
/* 只有当 body 有这个 class 时才生效 */
body.enable-mobile-scale #root {
  transform: scale(0.75);  /* 根据设备自动调整 */
}
```

## 🚀 如何使用

### 用户端
1. 在移动设备上访问 `www.s-l.ai`
2. 点击右下角的浮动按钮
3. 选择"启用缩放"或"禁用缩放"
4. 设置自动保存

### 开发端
```typescript
import { 
  enableMobileScale,    // 启用
  disableMobileScale,   // 禁用
  toggleMobileScale,    // 切换
  getDeviceInfo,        // 获取设备信息
  logDeviceInfo         // 调试信息
} from './utils/mobileScaleDetector';

// 启用缩放
enableMobileScale();

// 获取设备信息
const info = getDeviceInfo();
console.log(info.recommendedScale); // 0.75
```

## 🔧 自定义缩放比例

编辑 `src/styles/mobile-scale.css`:

```css
/* 修改大屏手机的缩放比例 */
body.enable-mobile-scale {
  @media (min-width: 481px) and (max-width: 768px) and (orientation: portrait) {
    --mobile-scale-ratio: 0.8;  /* 改为 0.8 */
  }
}
```

## 📞 常见问题

### Q: 如何完全禁用?
A: 从 `main.tsx` 移除 `import "./styles/mobile-scale.css";`

### Q: 会影响现有样式吗?
A: 不会!完全独立,只在启用时生效。

### Q: 如何调整比例?
A: 编辑 `mobile-scale.css` 中的 `--mobile-scale-ratio`

### Q: 为什么有些元素没缩放?
A: 缩放只应用于 `#root` 内的内容。
