3. 识别哪些是重要的（需要在网页中显示）

#### 第 2 步：转换 WMF/EMF 图片

**方法 A：使用 Windows 画图工具**（简单但需要逐个转换）

```
1. 右键点击 .wmf 或 .emf 文件
2. 选择"打开方式" → "画图"
3. 文件 → 另存为 → PNG
4. 保存为 image_XXX_converted.png
```

**方法 B：使用在线转换工具**（可批量转换）

```
1. 访问：https://convertio.co/zh/wmf-png/
2. 上传所有 .wmf 和 .emf 文件
3. 点击"转换"
4. 下载转换后的 PNG 文件
```

**方法 C：使用 IrfanView**（专业工具，支持批量转换）

```
1. 下载 IrfanView：https://www.irfanview.com/
2. 打开 IrfanView
3. 文件 → 批量转换/重命名
4. 添加所有 .wmf 和 .emf 文件
5. 输出格式选择 PNG
6. 点击"开始批量转换"
```

#### 第 3 步：在 Word 中替换图片

```
1. 打开 Word 文档：202509290462(1).docx
2. 找到每张 WMF/EMF 图片（通常是公式或图表）
3. 删除原图片
4. 插入 → 图片 → 选择转换后的 PNG 文件
5. 调整大小和位置
6. 重复以上步骤，替换所有 16 张图片
7. 保存文档
```

#### 第 4 步：重新上传

