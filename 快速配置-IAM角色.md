# 快速配置 EC2 访问 S3 - IAM 角色方法

## 第一步：创建 IAM 角色（5分钟）

### 1. 打开 IAM 控制台
🔗 直接访问：https://console.aws.amazon.com/iam/home#/roles

### 2. 创建角色
1. 点击橙色按钮 **"Create role"**（创建角色）
2. 在 "Select trusted entity" 页面：
   - 选择 **"AWS service"**
   - 选择 **"EC2"**
   - 点击 **"Next"**

### 3. 添加权限
1. 在搜索框输入：`S3`
2. 勾选 **"AmazonS3FullAccess"**
3. 点击 **"Next"**

### 4. 命名角色
1. Role name（角色名称）：`EC2-S3-Access`
2. 点击 **"Create role"**（创建角色）

✅ 角色创建完成！

---

## 第二步：将角色附加到 EC2 实例（2分钟）

### 1. 打开 EC2 控制台
🔗 直接访问：https://console.aws.amazon.com/ec2/home#Instances:

### 2. 找到您的 EC2 实例
- 查找 IP 地址为 **18.221.125.254** 的实例
- 或者查找名称包含 "sl-news" 的实例

### 3. 附加 IAM 角色
1. **选中**您的 EC2 实例（点击左侧的复选框）
2. 点击顶部的 **"Actions"**（操作）按钮
3. 选择 **"Security"** → **"Modify IAM role"**
4. 在下拉菜单中选择 **"EC2-S3-Access"**
5. 点击 **"Update IAM role"**（更新 IAM 角色）

✅ 角色附加完成！
