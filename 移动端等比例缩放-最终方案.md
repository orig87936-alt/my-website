## 🔧 技术实现

### 核心文件

1. **`src/styles/mobile-proportional.css`** - 等比例缩放样式
2. **`src/main.tsx`** - 引入缩放样式
3. **`src/components/MobileScaleToggle.tsx`** - 缩放开关（可选）

### 工作原理

```css
/* 在移动端 (<1280px) 自动应用缩放 */
@media (max-width: 1280px) {
  #root {
    transform: scale(var(--mobile-scale-ratio));
    transform-origin: top center;
    width: calc(100% / var(--mobile-scale-ratio));
  }
}
```

### 缩放比例配置

```css
/* 小屏手机 */
@media (max-width: 480px) and (orientation: portrait) {
  :root {
    --mobile-scale-ratio: 0.65;  /* 65% 缩放 */
  }
}

/* 大屏手机 */
@media (min-width: 481px) and (max-width: 768px) and (orientation: portrait) {
  :root {
    --mobile-scale-ratio: 0.75;  /* 75% 缩放 */
  }
}
```

